Rainbow Agenda

Privacy

Registered Users

In order to register and be able to use Rainbow Agenda, an unregistered user is asked to provide email and password. The e-mails are used by Rainbow Agenda only for the purpose of having authorized accounts and emails for sharing the meeting notes. We encrypt that passwords and if a registered user forgets his password, the only way to gain back access is to use the Forget password option.

Google Analytics

Rainbow Agenda uses Google Analytics, which is a Google tool for web analytics, to collect and analyze the user behaviour in order to optimize the services provided to the registered users.

Local Storage

Rainbow Agenda uses the local storage of a browser to collect information locally which is used to speed up the website time load. Each user may use the “Remember me” option and he will be able to access the website without the need to register. We advise the users to use this option only on their personal computers. If the website is accessed via a public computer, we advise the users to log out after they stop using the website so that other users of that public computer will not gain access to unauthorized private data.

Authorisation

Rainbow Agenda uses security techniques to prevent unauthorized access to your account or meeting. Each user has to authorize himself with email and password in order to get logged in.
